# Changelog

## 2026-02-17

### Добавлено

- Добавлен пожизненный кэш справочника эмитентов (`SECID -> EMITTER_ID`, `EMITTER_ID -> ISSUER_NAME`), чтобы не запрашивать повторно неизменяемые данные на каждом запуске.
- Добавлен механизм чекпоинтов для этапа обогащения эмитентов: прогресс сохраняется пакетами и может быть восстановлен после обрыва.
- Реализован п.4 из Improvement: пакетная обработка SECID и эмитентов с периодическим сохранением прогресса в локальный кэш.
- Реализовано инкрементальное обновление: при наличии кэша свежие данные объединяются с уже сохранёнными, обновляя изменённые записи и добавляя новые.
- Добавлен лист `INFO` в итоговый Excel с понятной расшифровкой полей для пользователей без технической подготовки.
- Добавлено автоформатирование Excel: заморозка шапки, фильтры, оформление заголовков и автоширина колонок.
- Добавлен замер времени выполнения скрипта в конце каждого запуска.
- Добавлена колонка `ISSUER_NAME` с наименованием эмитента облигации.
- Добавлено правило перестановки: колонка `ISIN` выводится первой в итоговой таблице.
- Добавлена колонка `ISSUER_INN` (ИНН эмитента) и сохранение ИНН в кэше эмитентов для повторных запусков.
- Реализован п.2 из Improvement: автоматическая проверка качества данных перед выгрузкой (пустые ISIN, дубликаты, аномалии купона) с отчётом в лог.

### Улучшено
- Обновлён визуальный стиль листа `MOEX_BONDS`: скрыта стандартная сетка, добавлена "зебра" строк и форматирование числовых полей.
- Добавлена параллельная обработка для обогащения данных эмитентов, чтобы выгрузка не выглядела зависшей и выполнялась стабильнее.

### Изменено
- Колонка `ISSUER_NAME` теперь располагается сразу после `ISIN`.
- Для колонки `ISSUER_NAME` увеличена ширина в Excel, чтобы длинные названия эмитентов читались без ручной правки.
- Из итогового файла удалены колонки `SECNAME`, `LISTLEVEL`, `STATUS`.
- Колонка `SECID` сохранена для технической работы, но скрыта в листе `MOEX_BONDS`.
- Улучшены комментарии этапов выполнения в логах, чтобы было видно прогресс работы.
