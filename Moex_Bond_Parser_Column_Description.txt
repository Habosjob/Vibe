SECID — Уникальный биржевой тикер (код) ценной бумаги.

SHORTNAME — Краткое наименование бумаги для отображения в интерфейсах.

NAME — Полное официальное наименование облигации.

TYPENAME — Название типа ценной бумаги (например, "Корпоративная облигация").

ISIN — Международный идентификационный номер ценной бумаги.

REGNUMBER — Государственный регистрационный номер выпуска.

LISTLEVEL — Уровень листинга на бирже (1, 2 или 3).

FACEVALUE — Номинальная стоимость одной облигации.

FACEUNIT — Валюта номинала (например, RUB, USD).

ISSUESIZE — Общий объем эмиссии (сумма или количество).

IS_COLLATERAL — Флаг, является ли бумага обеспечением (например, для РЕПО).

IS_EXTERNAL — Флаг, является ли бумага внешней (например, еврооблигацией).

PRIMARY_BOARDID — Идентификатор основной торговой площадки (режима).

PRIMARY_BOARD_TITLE — Название основной торговой площадки.

MATDATE — Дата погашения облигации.

IS_RII — Флаг доступности для неквалифицированных (розничных) инвесторов.

INCLUDEDBYMOEX — Флаг включения в официальные индексы или списки Московской биржи.

IS_QUALIFIED_INVESTORS — Флаг, что бумага предназначена только для квалифицированных инвесторов.

HIGH_RISK — Флаг отнесения инструмента к высокорисковым.

EVENINGSESSION — Флаг доступности торгов в вечерней сессии.

MORNINGSESSION — Флаг доступности торгов в утренней сессии.

WEEKENDSESSION — Флаг доступности торгов в сессию выходного дня.

BOND_TYPE — Основной тип облигации.

BOND_SUBTYPE — Подтип облигации.

SUSPENSION_LISTING — Флаг приостановки торгов или листинга.

ZSPREAD — Z-спред (разница в доходности относительно безрисковой кривой).

WAPRICE — Средневзвешенная цена за торговую сессию.

YIELDATWAP — Доходность к погашению, рассчитанная по средневзвешенной цене.

ISSUEDATE — Дата выпуска (эмиссии) облигации.

REGISTRYCLOSEDATE — Дата закрытия реестра для выплаты купона.

DIVIDENDVALUE — Абсолютный размер купонного платежа.

DIVIDENDYIELD — Купонная доходность (в процентах годовых).

REGISTRYCLOSETYPE — Тип закрытия реестра.

EMITENTNAME — Полное наименование компании-эмитента.

INN — ИНН эмитента.

LOTSIZE — Количество бумаг в одном стандартном лоте.

PRICE — Текущая рыночная цена (в валюте номинала).

PRICE_RUB — Текущая рыночная цена, пересчитанная в рубли.

RTL1 — Доходность к погашению? (RTL — часто Return to Low/Level).

RTH1 — Доходность к оферте? (RTH — часто Return to High).

RTL2 — Альтернативный вид доходности (уточнение требуется).

RTH2 — Альтернативный вид доходности (уточнение требуется).

RTL3 — Альтернативный вид доходности (уточнение требуется).

RTH3 — Альтернативный вид доходности (уточнение требуется).

DISCOUNT1 — Уровень дисконта №1 для маржинальной торговли.

LIMIT1 — Лимит №1, связанный с дисконтом.

DISCOUNT2 — Уровень дисконта №2.

LIMIT2 — Лимит №2.

DISCOUNT3 — Уровень дисконта №3.

DISCOUNTL0 — Нижний (Low) уровень дисконта для параметров РЕПО.

DISCOUNTH0 — Верхний (High) уровень дисконта для параметров РЕПО.

FULLCOVERED — Флаг, указывающий на полное покрытие (например, для сделок РЕПО).

FULL_COVERED_LIMIT — Лимит, применяемый при полном покрытии.
