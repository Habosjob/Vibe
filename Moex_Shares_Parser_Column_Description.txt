SECID — Уникальный биржевой тикер (код) акции.

SHORTNAME — Краткое наименование акции для отображения.

NAME — Полное официальное наименование акции.

TYPENAME — Тип акции (например, "Акция обыкновенная", "Акция привилегированная").

ISIN — Международный идентификационный номер ценной бумаги.

REGNUMBER — Государственный регистрационный номер выпуска акций.

LISTLEVEL — Уровень листинга на бирже.

FACEVALUE — Номинальная стоимость одной акции.

FACEUNIT — Валюта номинала (например, RUB).

ISSUESIZE — Общее количество акций в выпуске.

IS_COLLATERAL — Флаг, может ли акция использоваться в качестве обеспечения.

IS_EXTERNAL — Флаг, является ли бумага внешней.

PRIMARY_BOARDID — Идентификатор основной торговой площадки.

PRIMARY_BOARD_TITLE — Название основной торговой площадки.

MATDATE — Для акций поле обычно пустое или не используется (акции бессрочные).

IS_RII — Флаг доступности для неквалифицированных инвесторов.

INCLUDEDBYMOEX — Флаг включения в индексы Московской биржи (IMOEX и др.).

IS_QUALIFIED_INVESTORS — Флаг доступности только для квалифицированных инвесторов.

HIGH_RISK — Флаг отнесения к высокорисковым активам.

EVENINGSESSION — Флаг торгов в вечерней сессии.

MORNINGSESSION — Флаг торгов в утренней сессии.

WEEKENDSESSION — Флаг торгов в сессию выходного дня.

BOND_TYPE — Для акций поле не несёт смысла, скорее всего пустое.

BOND_SUBTYPE — Для акций поле не несёт смысла, скорее всего пустое.

SUSPENSION_LISTING — Флаг приостановки торгов/листинга.

ZSPREAD — Для акций не рассчитывается, поле пустое.

WAPRICE — Средневзвешенная цена за сессию.

YIELDATWAP — Для акций может означать дивидендную доходность по средневзвешенной цене.

ISSUEDATE — Дата выпуска (эмиссии) акций.

REGISTRYCLOSEDATE — Дата закрытия реестра для выплаты дивидендов.

DIVIDENDVALUE — Объявленный размер дивиденда на одну акцию.

DIVIDENDYIELD — Дивидендная доходность (в процентах).

REGISTRYCLOSETYPE — Тип закрытия реестра (под дивиденды).

EMITENTNAME — Полное наименование компании-эмитента.

INN — ИНН эмитента.

LOTSIZE — Количество акций в одном стандартном лоте.

PRICE — Текущая рыночная цена (последняя сделка).

PRICE_RUB — Текущая цена в рублях (актуально для иностранных акций).

RTL1 — Показатель доходности/риска 1 (уточняется по спецификации MOEX).

RTH1 — Показатель доходности/риска 2 (уточняется по спецификации MOEX).

RTL2 — Показатель доходности/риска 3 (уточняется по спецификации MOEX).

RTH2 — Показатель доходности/риска 4 (уточняется по спецификации MOEX).

RTL3 — Показатель доходности/риска 5 (уточняется по спецификации MOEX).

RTH3 — Показатель доходности/риска 6 (уточняется по спецификации MOEX).

DISCOUNT1 — Уровень дисконта №1 (для маржинальной торговли).

LIMIT1 — Лимит №1.

DISCOUNT2 — Уровень дисконта №2.

LIMIT2 — Лимит №2.

DISCOUNT3 — Уровень дисконта №3.

DISCOUNTL0 — Нижний уровень дисконта для параметров РЕПО/кредитования.

DISCOUNTH0 — Верхний уровень дисконта для параметров РЕПО/кредитования.

FULLCOVERED — Флаг полного покрытия (для сделок РЕПО).

FULL_COVERED_LIMIT — Лимит для операций с полным покрытием.
