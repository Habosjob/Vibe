# MONITORING

## Логи
- `logs/latest.log` перезаписывается при каждом запуске.
- Уровень логирования задается в `config/config.yml -> logging.level`.

## Этапы
Во время запуска в консоль и лог выводятся этапы, чтобы процесс не выглядел зависшим:
1. Загрузка источников.
2. Очистка и нормализация.
3. Скоринг и фильтрация.
4. Экспорт в Excel.

## Сводка запуска
В конце запуска выводится:
- сколько бумаг обработано;
- сколько отфильтровано;
- сколько ошибок;
- общее время выполнения.
