# Мониторинг и логи

## Логи
- Файл: `logs/latest.log`
- Перезаписывается при каждом запуске.
- Уровни:
  - `INFO` — этапы работы.
  - `WARNING/ERROR` — проблемы API и обработки.

## Что смотреть при проблемах
1. Наличие записей `Ошибка запроса`.
2. Количество ошибок в финальной сводке.
3. Проверить интернет-соединение и доступ к `iss.moex.com`.
