# Структура проекта

- `main.py` — точка входа основного пайплайна (остается в корне).
- `generate_raw_links.py` — утилита генерации ссылок.
- `cache/` — кэш и снапшоты источников (включая просроченные данные для fallback).
- `logs/` — перезаписываемые технические логи скриптов.
- `output/` — итоговые Excel-файлы выгрузок.
- `docs/` — документация и служебные текстовые артефакты.
