# MERGE_BONDS.py — склейка Moex и Dohod по ISIN

## Что делает скрипт
`MERGE_BONDS.py` берет `Moex_Bonds.xlsx` как базовый набор и добавляет к нему поля из `Dohod_Bonds.xlsx` **только для тех бумаг, которые уже есть в Moex**.

Сопоставление выполняется строго по `ISIN` (left join от Moex).

В итоговый файл `BondsFinal.xlsx` добавляются колонки:
- `Дата` (из `Ближайшая дата погашения/оферты (Дата)` или алиасов);
- `Текущий номинал`;
- `Тип купона`;
- `Субординированная (да/нет)`;
- `Базовый индекс (для FRN)`;
- `Премия/Дисконт к базовому индексу (для FRN)`.

## Нормализация даты
Колонка `Дата` приводится к datetime-типу (`datetime64[ns]`) и при сохранении в Excel получает формат `dd.mm.yyyy`.
Это позволяет корректно фильтровать даты в Excel как даты, а не как строки.

## Форматирование Excel
Сохранение делается через тот же механизм, что и в `Moex_Bonds.py` (`save_to_excel`), поэтому визуальный стиль совпадает:
- табличный стиль;
- фиксированный заголовок;
- автофильтр;
- полосатая заливка;
- числовые/датовые форматы;
- цветовая шкала по колонкам из `output.heatmap_columns`.

## Интерактивный вывод
Скрипт работает с прогресс-баром и анимированными сообщениями в консоли, чтобы пользователь видел, что процесс выполняется.

## Логи
Отдельный перезаписываемый лог:
- `logs/MERGE_BONDS.log`

В лог пишутся:
- размеры входных таблиц;
- какие колонки из Dohod были сопоставлены;
- сколько строк сопоставилось по ISIN;
- сколько дат успешно распознано;
- ошибки с трассировкой.

## Настройки (YAML)
Все настройки — в `config/moex_bonds.yaml`, секция `merge_bonds`:

```yaml
merge_bonds:
  input:
    moex_excel_path: "Moex_Bonds.xlsx"
    moex_sheet_name: "MOEX_BONDS"
    dohod_excel_path: "Dohod_Bonds.xlsx"
    dohod_sheet_name: null
  output:
    excel_path: "BondsFinal.xlsx"
    sheet_name: "BONDS_FINAL"
  logging:
    path: "logs/MERGE_BONDS.log"
  dohod_columns:
    "Дата":
      - "Ближайшая дата погашения/оферты (Дата)"
      - "Дата погашения/оферты"
      - "Дата"
    ...
```

### Что можно настраивать
- входные/выходные файлы и листы;
- путь к логу;
- алиасы колонок из Dohod (`dohod_columns`) — полезно, если на сайте меняются названия столбцов.

## Запуск
```bash
python3 MERGE_BONDS.py --config config/moex_bonds.yaml
```
