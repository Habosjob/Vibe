# Скоринг

Текущая версия не применяет скоринг и фильтрацию. Поле `Отфильтровано` в сводке = 0.

План: добавить базовый скоринг по ликвидности, листингу и доходности в следующих итерациях.


## Облегченная модель скоринга
- В `output/emitents.xlsx` добавлены столбцы `Scorerate` и `DateScore`.
- Допустимые значения `Scorerate`: `Blacklist`, `Redlist`, `Yellowlist`, `Greenlist` (в Excel включена валидация ввода).
- Если `HASDEFAULT=1`, эмитент автоматически помечается как `Blacklist`.
- `DateScore` обновляется датой запуска, когда значение `Scorerate` изменилось относительно прошлого состояния.
- В итоговом `MOEX_BONDS` перед `SHORTNAME` выводится столбец `ScoreColor`: `🟢/🟡/🔴` для `Greenlist/Yellowlist/Redlist`; для пустого скоринга — пусто.
- Бумаги эмитентов со статусом `Blacklist` исключаются из итогового файла и не обогащаются дополнительно на следующих прогонах.
