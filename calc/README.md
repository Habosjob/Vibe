# YTM калькулятор (штучный, интерактивный)

## Что делает скрипт
`calc/ytm_calculator.py` — автономный монолитный скрипт для расчета доходностей по одной облигации за раз:
- просит ISIN в интерактивной консоли;
- сам подтягивает данные по бумаге из MOEX ISS (амортизация, даты, рыночные поля) и Corpbonds (тип купона, формула, ставка, НКД, цена, ближайшие даты);
- считает:
  - YTM (эффективная годовая),
  - простую доходность к погашению,
  - текущую доходность (ТКД) по ожидаемым выплатам ближайших 12 месяцев,
  - simple margin для флоатеров как `ТКД - текущий индекс` (КС/RUONIA/G-Curve),
  - срок до ближайшего события (оферта/погашение, что раньше) в годах;
- поддерживает полуавтоматический режим: можно вручную поправить цену, НКД, частоту и тип купона, а для флоатера — выбрать индекс короткими подсказками `(r/c/g)` и премию.

## Как запустить
```bash
python calc/ytm_calculator.py
```

Далее введите ISIN в консоль. Для выхода: `exit`.

## Логирование и прогресс
- В консоль выводятся только этапы, прогресс-бары `tqdm` и итоговая таблица.
- Технические детали пишутся в перезаписываемый лог `logs/ytm_calc.log`.
- В конце выводится `Summary` по времени этапов и общего прогона.

## Настройка
Все настройки вынесены в начало файла `calc/ytm_calculator.py` в блок `CONFIG`.
У каждого флага есть комментарий (что делает, какие значения, дефолт).

Ключевые настройки:
- `KEY_RATE_FORECAST` — прогноз ключевой ставки для флоатеров.
- `INFLATION_FORECAST` — прогноз инфляции для линкеров.
- `ENABLE_MANUAL_OVERRIDE` — включение/выключение ручной корректировки.
- `YTM_OUTPUT_PRECISION` — точность вывода доходностей.
- `HTTP_TIMEOUT_SECONDS` — таймаут сетевых запросов.

## Ограничения текущей версии
- Формула купона и тип купона берутся приоритетно из Corpbonds (как в основном пайплайне), при недоступности Corpbonds используйте ручной ввод в полуавтоматическом шаге.
- Для zcyc-бенчмарков, если кривая недоступна, расчет использует fallback-логику.


## Ручная корректировка (новый UX)
- Тип купона: подсказки `(f)` флоатер, `(x)` фикс, `(o)` прочее.
- Для флоатера:
  - выбор индекса: `(r)` RUONIA, `(c)` КС ЦБ, `(g)` G-Curve;
  - если выбран `g`, запрашивается срок кривой (в годах);
  - отдельно запрашивается премия к индексу.
- Если поле не заполнено вручную, сохраняется дефолт из загруженных источников.
