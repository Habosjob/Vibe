# Сбор облигаций MOEX в Excel

## Что делает проект
Этот проект получает список всех облигаций, которые торгуются на Московской бирже (MOEX), и сохраняет результат в удобный Excel-файл.

Скрипт рассчитан на простой запуск в VS Code: без аргументов командной строки и сложной настройки.

---

## Основной файл
- `moex_bonds_to_excel.py` — главный скрипт.

### Что делает скрипт
1. Подготавливает рабочие папки.
2. Очищает папку `raw` перед запуском.
3. Настраивает логирование в `logs/moex_bonds.log` (старый лог перезаписывается).
4. Пытается загрузить данные из кэша (`cache/moex_bonds_cache.json`).
5. Если кэш устарел или отсутствует — скачивает данные с MOEX ISS API.
6. Сохраняет «сырые» данные в `raw/moex_bonds_raw.json`.
7. Сохраняет итог в `output/moex_bonds.xlsx`.

---

## Структура папок
- `logs/` — лог последнего запуска.
- `raw/` — сырые данные для отладки (очищается автоматически перед каждым запуском).
- `cache/` — кэш API-ответов.
- `output/` — готовый Excel-файл.

---

## Как запустить
Откройте проект в VS Code и выполните:

```bash
python moex_bonds_to_excel.py
```

После выполнения проверьте файл:
- `output/moex_bonds.xlsx`

---
