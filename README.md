# Сбор облигаций MOEX в Excel

## Что делает проект
Этот проект получает список облигаций с Московской биржи (MOEX) и формирует удобный Excel-файл для ежедневной работы.

Скрипт рассчитан на простой запуск в VS Code: **без аргументов командной строки**.

---

## Основной файл
- `moex_bonds_to_excel.py` — главный скрипт выгрузки и подготовки отчёта.

### Что делает скрипт по шагам
1. Создаёт служебные папки (`logs`, `raw`, `cache`, `output`).
2. Полностью очищает папку `raw` перед запуском.
3. Перезаписывает лог `logs/moex_bonds.log` (лог всегда только за последний запуск).
4. Пытается прочитать кэш.
5. Запрашивает свежие данные с MOEX.
6. Выполняет инкрементальное обновление (добавляет новые записи и обновляет изменившиеся).
7. Сохраняет сырые данные в `raw/moex_bonds_raw.json`.
8. Обогащает данные: добавляет наименование эмитента облигации (`ISSUER_NAME`).
9. Формирует `output/moex_bonds.xlsx` с улучшенным форматированием и листом `INFO`.
10. В конце пишет общее время выполнения.

## Структура папок
- `logs/` — лог последнего запуска.
- `raw/` — сырые данные для отладки (очищается автоматически перед каждым запуском).
- `cache/` — кэш API-ответов.
- `output/` — итоговый Excel-файл.

---

## Как запустить
Откройте проект в VS Code и выполните:

```bash
python moex_bonds_to_excel.py
```

После завершения проверьте:
- `output/moex_bonds.xlsx`
- `logs/moex_bonds.log`
