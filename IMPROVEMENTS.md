# Improvement backlog

## 1) История изменений по выпускам облигаций (отдельный лист HISTORY)
- **Задача:** Накапливать изменения ключевых полей (купон, цена, статус торгов) и выводить их в отдельный лист `HISTORY`.
- **Зачем это нужно:** Позволит видеть динамику, а не только «срез на сейчас».
- **Польза простым языком:** Можно понять, что изменилось со вчерашнего дня, не сравнивая файлы вручную.
- **Степень полезности:** **5/5**

## 2) Подсветка новых и изменённых строк в Excel
- **Задача:** При сохранении Excel визуально отмечать новые/изменённые строки цветом.
- **Зачем это нужно:** Быстрое визуальное понимание изменений без ручного сравнения.
- **Польза простым языком:** Открыл файл — сразу видно, что новое и что поменялось.
- **Степень полезности:** **4/5**

## 3) Загрузка всех страниц MOEX ISS (не только первой)
- **Задача:** Сделать постраничную загрузку всех выпусков, пока API возвращает данные.
- **Зачем это нужно:** Сейчас при больших выборках можно получить неполный набор бумаг.
- **Польза простым языком:** В отчёт будут попадать все доступные облигации, а не только первая часть.
- **Степень полезности:** **5/5**
