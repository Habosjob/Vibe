# IMPROVEMENTS

## Идея 10
- **Задача:** Добавить fallback-источник ключевой ставки ЦБ (второй URL + выбор первого успешного).
- **Зачем:** Один внешний API может быть временно недоступен, из-за чего плавающий купон считается неполно.
- **Профит простыми словами:** Купон почти всегда будет считаться правильно даже при проблемах с одним источником.
- **Полезность 1..5:** 5
