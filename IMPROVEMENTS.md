# Идеи для улучшений

## 1) Автоматическая проверка качества данных перед сохранением в Excel
- **Задача:** добавить проверку на пустые ключевые поля (например, `SECID`, `EMITENT_ID`, `ISIN`) и вывод понятного отчета.
- **Зачем это нужно:** чтобы сразу видеть, если биржа вернула неполные данные и избежать ошибок в аналитике.
- **Профит простыми словами:** меньше риска принять неверные решения на «битых» данных.
- **Степень полезности:** 5/5.

## 2) Разделение итогового Excel на несколько файлов по типам облигаций
- **Задача:** сохранять отдельные Excel-файлы для ОФЗ, корпоративных и замещающих облигаций.
- **Зачем это нужно:** упростить работу пользователям, которым нужен только один сегмент.
- **Профит простыми словами:** файл открывается быстрее, легче ориентироваться в данных.
- **Степень полезности:** 4/5.

## 3) Планировщик ежедневного запуска
- **Задача:** добавить простой bat/sh скрипт для ежедневного запуска по расписанию.
- **Зачем это нужно:** чтобы данные обновлялись автоматически без ручного запуска.
- **Профит простыми словами:** экономия времени и всегда свежий Excel под рукой.
- **Степень полезности:** 4/5.
