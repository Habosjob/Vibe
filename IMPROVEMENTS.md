# IMPROVEMENTS

## Идея 11
- **Задача:** Добавить автоматическую подсказку по lock-файлам (`*.lock`) в конце запуска (`что найдено` / `как снять`).
- **Зачем:** Пользователь может забыть удалить lock-файл и не понять, почему изменился режим.
- **Профит простыми словами:** Меньше «магии», проще поддержка без погружения в код.
- **Полезность 1..5:** 4

## Идея 12
- **Задача:** Добавить Excel-лист `EXCLUSIONS_HISTORY` с историей исключений из `state/exclusions_history.json`.
- **Зачем:** Сейчас история есть в JSON, но пользователю удобнее смотреть её в привычном Excel.
- **Профит простыми словами:** История причин исключений становится доступной в 2 клика.
- **Полезность 1..5:** 5

## Идея 13
- **Задача:** Добавить режим `run.py` «только отчёт» без сетевых запросов (на основе сохранённых state + runs).
- **Зачем:** Для быстрой проверки результатов не всегда нужен полный повторный парсинг MOEX.
- **Профит простыми словами:** Можно мгновенно получить сводку и файлы без ожидания сети.
- **Полезность 1..5:** 4

## Идея 14

- **Зачем:** Пустые ИНН/наименования лучше отслеживать сразу после запуска, а не в ручной проверке.
- **Профит простыми словами:** Быстро видно, когда источник данных «просел» по качеству.
- **Полезность 1..5:** 5
