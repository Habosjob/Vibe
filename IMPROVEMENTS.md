# Improvement backlog

## 0) Отчёт по источнику формулы купона (DOHOD/CORPBONDS)
- **Задача:** Добавить в итоговый Excel отдельное поле/лист, где видно, из какого источника была получена формула для `COUPONPERCENT`.
- **Зачем это нужно:** Сейчас формула может прийти из двух источников, но пользователь видит только итоговую ставку.
- **Польза простым языком:** Можно быстро понять, откуда взялась цифра купона для конкретной бумаги, без чтения логов.
- **Степень полезности:** **4/5**

## 1) История изменений по выпускам облигаций (отдельный лист HISTORY)

- **Польза простым языком:** Можно понять, что изменилось со вчерашнего дня, не сравнивая файлы вручную.
- **Степень полезности:** **5/5**

- **Польза простым языком:** Открыл файл — сразу видно, что новое и что поменялось.
- **Степень полезности:** **4/5**
