# Improvement backlog

## 1) Инкрементальное обновление данных вместо полной перезагрузки
- **Задача:** Загружать только новые или изменённые облигации, а не весь список целиком.
- **Зачем это нужно:** Ускорит работу скрипта и снизит нагрузку на API MOEX.
- **Польза простым языком:** Скрипт будет запускаться быстрее и реже «тормозить» при плохом интернете.
- **Степень полезности:** **5/5**

## 2) Автоформат Excel (ширина колонок, фильтры, заморозка шапки)
- **Задача:** Добавить визуальное форматирование итогового Excel.
- **Зачем это нужно:** Чтобы таблица сразу была удобна для чтения и фильтрации.
- **Польза простым языком:** Файл можно сразу открывать и работать, без ручной «косметики».
- **Степень полезности:** **4/5**

## 3) Отдельный лист со справкой по полям в Excel
- **Задача:** Добавить лист `INFO` с расшифровкой колонок (что означает SECID, ISIN и т.д.).
- **Зачем это нужно:** Неподготовленным пользователям проще понять содержимое файла.
- **Польза простым языком:** Даже человек без опыта в IT и биржевых данных быстрее разберётся в отчёте.
- **Степень полезности:** **4/5**
